# Q+A RSS Generator

Scrapes ABC's website and builds a RSS feed containing Q+A episodes.

## Usage
```bash
./update.py
```

## Notes

`scrape.py`, `add-field.py`, and `gen-feed.py` are for debugging purposes only.

```bash
./scrape.py | ./gen-feed.py > qanda.rss
```
